PROJECT TITLE: Forest Monkey
PURPOSE OF PROJECT: Learn to create game from scratch
VERSION 2.0
HOW TO START THIS PROJECT: 	Click RUN
							Select "Start Game"
AUTHORS: Benjamin Presley
USER INSTRUCTIONS:
Collect all the bananas without getting eaten to win.

Use "UP", "DOWN", "LEFT", "RIGHT" keys to move
